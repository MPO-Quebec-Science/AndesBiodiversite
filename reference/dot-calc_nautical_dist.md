# Calculate nautical distance between two geographic points \#' This function calculates the great-circle distance between two points on the Earth's surface \#' given their latitude and longitude using the Haversine formula.

Calculate nautical distance between two geographic points \#' This
function calculates the great-circle distance between two points on the
Earth's surface \#' given their latitude and longitude using the
Haversine formula.

## Usage

``` r
.calc_nautical_dist(p1_lat, p1_lon, p2_lat, p2_lon)
```

## Arguments

- p1_lat:

  Latitude of the first point in decimal degrees

- p1_lon:

  Longitude of the first point in decimal degrees

- p2_lat:

  Latitude of the second point in decimal degrees

- p2_lon:

  Longitude of the second point in decimal degrees
